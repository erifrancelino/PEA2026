# Projeto Jogos Acessiveis

Projeto colaborativo para desenvolvimento de funcionalidades de acessibilidade em jogos.

Arquivos do projeto:
- novo.py
- nova_funcionalidade.py
- jogo_acessivel.py


# Sistema de Jogos Acessíveis

Projeto desenvolvido na disciplina Projeto de Software – Unidade 4.

## Objetivo
Aplicar conceitos de Gerenciamento de Configuração utilizando Git e GitHub em um ambiente colaborativo.

## Funcionalidades
- Criação e modificação de artefatos
- Controle de versões
- Atualização de repositório remoto
- Registro de histórico de alterações

## Tecnologias utilizadas
- Python
- Git
- GitHub
